概述

FEG 是一个基于 gulp/gulp-plugins 的前端开发集成解决方案。FEG 定义了一系列任务，这些自动执行的任务可以把你从繁杂无聊的复制、粘贴、合并、压缩、重命名、打包等事务中解脱出来，从而更专注于具体业务的开发实现。

插件

插件 描述
gulp-sass SASS文件编译
gulp-clean-css 压缩CSS
gulp.spritesmith 合并雪碧图
gulp-file-include 引用HTML模版
gulp-livereload 实时刷新
gulp-webserver 开启本地服务
gulp-zip 文件打包

结构

feg
|--include          //模版文件
|--static           //静态文件
   |--css           //样式文件
   |--img           //图片文件
      |--sprite     //雪碧图文件夹
         |--a       //雪碧图a组
         |--b       //雪碧图b组
         |--c       //雪碧图c组
   |--js            //脚本文件
      |--app        //项目脚本
      |--libs       //第三方脚本

安装使用前请安装Node.js

//安装cnpm
npm install -g cnpm

//进入feg目录安装插件
cnpm install

使用

//使用帮助
gulp h

//启动任务
gulp s

//打包代码
gulp b

其他

关于雪碧图，这里做了按照文件夹分组合并的方式，具体使用方法请查看 gulpfile.js 代码中的 sprite 任务。如果图标有三态，命名请按照类似： xxx.png（默认），xxx-h.png（划过），xxx-v.png（点击）
